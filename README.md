# Port-Trade-Vision

Port-Trade-Vision is a web application developed to visualize and manage port trade operations efficiently.  
This project provides an interactive dashboard for monitoring shipment data, trade analytics, and logistics management.

---

## Features

- Interactive Dashboard
- Trade Data Visualization
- Shipment Monitoring
- Responsive User Interface
- Fast Performance using Vite
- TypeScript Support
- Modern UI using Tailwind CSS

---

## Technologies Used

- React
- TypeScript
- Vite
- Tailwind CSS
- Node.js
- Drizzle ORM
